Descripción:
-
Este repositorio contiene material de introducción al desarrollo backend, con carpetas separadas para `backend` y `frontend`. El objetivo es ofrecer una base mínima para practicar APIs, rutas, y despliegue local.

Estructura del repositorio
-
- `backend/` : Código del servidor (API, lógica, base de datos de ejemplo).
- `frontend/` : Interfaz de usuario (cliente) que consume la API del backend.
